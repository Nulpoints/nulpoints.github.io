ELiS Experimental Keyboard

Use shift and right-alt to access location and movement layers

Superscript is created by first typing (|) then the superscriped symbol. This key looks like "|" in ELiS

Underlining is done by first typing (') then the number 1, 2 or 3 depending on how many underlines wanted.

If you would like to use the keyboard on your phone or computer, the KMP file can be found here.
