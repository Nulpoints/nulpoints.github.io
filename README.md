Welcome to the official portal for **Nulpoints** sign language notation tools. This repository hosts a suite of interactive web applications and resources designed to help users read, write, study, and analyze three major sign language writing and description systems: **Escrita das Línguas de Sinais (ELiS)**, **SignFont (ASL)**, and **Sign Language Description (SLD)**.

---

## 🛠️ How to Enable Native Sign Notation Rendering

To view and type these notation systems natively on your desktop, mobile, or system applications, follow these simple steps:

> [!TIP]
> **Step 1: Install the Custom Fonts**
> 1. Download the custom font files:
>    - [SignFontNew.ttf](./SignFont/SignFontNew.ttf)
>    - [Elis2.ttf](./elis/Elis2.ttf)
> 2. Open each downloaded file on your computer and click **Install**. This allows your system to render the sign notation symbols instead of normal Latin characters.

> [!NOTE]
> **Step 2: Install Keyman Keyboard for Typing**
> 1. Download and install the free, open-source [Keyman Keyboard Manager](https://keyman.com/).
> 2. Download and double-click our keyboard bundle files to install them:
>    - [SignFont Keyman Keyboard (.KMP)](./SignFont/signfont.kmp)
>    - [ELiS Keyman Keyboard (.KMP)](./elis/elis.kmp)
> 3. You can now press standard keys to output the beautiful, custom sign notation visographemes in any document, browser, or chat window!
